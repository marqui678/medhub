
 Schilling test 
 
 
 
 To use the sharing features on this page, please enable JavaScript. 
 The Schilling can determine whether the body absorbs  vitamin B12  normally. It checks for certain types of anemia that are caused by a low vitamin B12 level. This test is rarely, if ever, done today. Alternative Names
